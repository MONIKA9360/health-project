# SUMO ❤️💙 - Daily Life Tracker for Moni & Surya

A romantic daily routine tracking app designed exclusively for couples to monitor their daily habits, health, and share private thoughts.

## Features

- 🔐 Secure login (sumo@gmail.com / sumo)
- ❤️💙 Split-screen dashboard for Moni (red theme) and Surya (blue theme)
- 📅 Complete daily routine tracking:
  - Morning, bath, travel, college, evening, and night routines
  - Sleep tracking with automatic hour calculation
  - Meal tracking with details
  - Water intake monitoring
- 💭 Private message box for sharing thoughts
- 📊 Health insights and analytics
- 📆 Calendar view with date history
- 🔥 Routine streak tracking
- 💾 Local storage persistence

## Installation

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```
